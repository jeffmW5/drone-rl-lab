# Memory -- Index

Read these files in order before every experiment:

1. **`memory/HARD_RULES.md`** -- Absolute constraints. Never violate.
2. **`memory/EXPERIMENT_LOG.md`** -- Full history of all experiments.
3. **`memory/INSIGHTS.md`** -- Kaggle targets, benchmark data, architecture notes.
4. **`memory/NEXT.md`** -- Prioritized research queue.

To regenerate the experiment log: `python compare.py --generate-log`
