# Memory -- Index

Read these files in order before every experiment:

1. **`memory/HARD_RULES.md`** -- Repo integrity and evaluation constraints only.
2. **`memory/EPISTEMIC_SCHEMA.md`** -- How claims get promoted, scoped, and demoted.
3. **`memory/BELIEF_AUDIT.md`** -- High-risk claims currently under review.
4. **`memory/FACTS.md`** -- Direct observations with sources.
5. **`memory/HYPOTHESES.md`** -- Active explanations being tested.
6. **`memory/TENTATIVE_LESSONS.md`** -- Scoped heuristics with confidence labels.
7. **`memory/EXPERIMENT_LOG.md`** -- Full history of all experiments.
8. **`memory/INSIGHTS.md`** -- Background references and benchmark context; not normative memory.
9. **`memory/NEXT.md`** -- Prioritized research queue.

To regenerate the experiment log: `python compare.py --generate-log`
