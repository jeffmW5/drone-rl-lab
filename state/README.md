# State Artifacts

`state/current.json` is the machine-readable snapshot of the lab's current state.

It is generated by:

```bash
python3 scripts/lab_state.py
```

The goal is to give autonomous agents one canonical place to read:

- current queue state
- active agents
- best known results
- repo provenance

Human-facing docs like `README.md`, `outbox/STATUS.md`, and `memory/*` can still
add interpretation, but the state snapshot is the source of truth for automation.
